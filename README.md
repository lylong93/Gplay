## use
add css
``` html
 <link rel="stylesheet" href="./dist/player.css">
```
add Gplay
``` html
 <script type="text/javascript" src="./dist/player.min.js"></script>
```
or
 ```js

  import Gplay from 'Gplay'
 ```
